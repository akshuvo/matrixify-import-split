# Matrixify Shopify Import File Split

This repository contains local Node.js utilities to process, split, and generate files (CSV and Excel) for Shopify Matrixify imports. It includes scripts to split large import files into smaller chunks of 10 items per file, as well as tools to extract image URLs from HTML and format them into Matrixify-compatible CSVs.

## Directory Structure

- `csv-split/`: Scripts for processing CSV files and extracting images.
- `xlxs-split/`: Scripts for processing Excel (.xlsx) files.

---

## 1. CSV Processing (`csv-split/`)

### Setup
Navigate to the `csv-split` directory and install the required dependencies (`csv-parse`, `csv-stringify`):

```bash
cd csv-split
npm install
```

### Scripts

#### A. Split CSV (`split.js`)
Reads an `input.csv` file, reverses the order of the rows (Z → A), and splits them into smaller chunked CSV files containing 10 rows each. The output files are named `output_1.csv`, `output_2.csv`, etc.

**Usage:**
```bash
node split.js
```
*(Ensure you have an `input.csv` file in the same directory before running).*

#### B. Extract Images from HTML (`extract_img.js`)
Parses a `demo.html` file to extract Shopify CDN image URLs (`domain.com/cdn/shop/files/...`). It cleans the URLs by removing query strings and resize modifiers, removes duplicate URLs, and saves the unique main URLs to `image_urls.txt`.

**Usage:**
```bash
node extract_img.js
```
*(Ensure valid paths are set in the script for the source HTML and target text files).*

#### C. Generate Matrixify Image CSV (`make_csv.js`)
Reads the `image_urls.txt` file generated by the extraction script and formats it into a Matrixify-compatible CSV file (`output_hp.csv`) that can be used for bulk uploading images. It automatically maps properties like ID, File Name, Command (`MERGE`), Link, Type (`IMAGE`), and Mime Type based on the extraction.

**Usage:**
```bash
node make_csv.js
```
*(Ensure valid paths are set in the script for the source user URL list and output CSV).*

## 2. Excel (XLSX) Processing (`xlxs-split/`)

### Setup
Navigate to the `xlxs-split` directory and install the required dependencies (`xlsx`):

```bash
cd xlxs-split
npm install
```

### Scripts

#### A. Split XLSX (`split.js`)
Reads an `input.xlsx` file from the first sheet, reverses the order of the rows (Z → A), and splits them into smaller chunked Excel files containing 10 rows each. The output files are named `output_1.xlsx`, `output_2.xlsx`, etc.

**Usage:**
```bash
node split.js
```
*(Ensure you have an `input.xlsx` file in the same directory before running).*
